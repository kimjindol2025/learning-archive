# 부트스트랩 인터프리터 상태 (2026-03-02)

## ✅ 현재 기능

### 데이터 타입
- ✅ 숫자 (number)
- ✅ 문자열 (string)

### 연산
- ✅ 산술: `+`, `-`, `*`, `/`, `%`
- ✅ 비교: `==`, `!=`, `<`, `>`

### 변수
- ✅ 할당: `x = 5`
- ✅ 재할당: `x = x + 1`

### 함수 (내장)
- ✅ `print()` - 출력
- ✅ `len()` - 길이

### 코드 실행
- ✅ 파일 실행: `node dist/index.js file.fl`
- ✅ eval 모드: `node dist/index.js --eval "1+2"`
