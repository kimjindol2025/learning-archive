## 📊 구현 계획

### Step 1: StandardLibrary (300줄)

**기본 함수들**:
```
print(value) → 값 출력
input() → 입력 받기
strlen(str) → 문자열 길이
strcmp(s1, s2) → 문자열 비교
parseInt(str) → 문자 → 정수
toString(num) → 정수 → 문자
```

### Step 2: RuntimeSupport (200줄)

**런타임 환경**:
```
프로그램 초기화
메인 함수 호출
결과 반환
메모리 정리
```

### Step 3: 실제 프로그램 (300줄)

**5개 예제 프로그램**:

