## 🏗️ Phase B 전체 구현

### Week 1: HashMap + Vector (완료)

**구현 내용**:
- `HashMap<K,V>`: 제네릭 해시 맵
  - 체이닝 충돌 처리
  - 동적 리사이징 (로드팩터 > 0.75)
  - O(1) 평균 조회 시간
  - 16개 메서드

- `Vector<T>`: 동적 배열
  - push/pop/get/set
  - map/filter 함수형 메서드
  - 제네릭 타입 지원

**테스트**: 12/12 통과 ✅

### Week 2: File I/O 시스템 (완료)

**구현 내용**:
- `FileHandle`: 파일 작업
  - 4가지 모드 (r, w, a, rw)
  - read/readLines/write/append
  - 파일 메타데이터 조회
  - 11개 메서드

- `Directory`: 디렉토리 관리
  - create/delete (재귀)
  - list/listFiles/listDirectories
  - 9개 메서드

- `FileSystem`: 유틸리티
  - copy/move/exists
  - isFile/isDirectory
  - 경로 정보 조회
  - 12개 정적 메서드

**테스트**: 12/12 통과 ✅

### Week 3: Path 유틸리티 (완료)

**구현 내용**:
- Path 클래스: 경로 처리
  - normalize: 경로 정규화
  - split/join: 경로 분해/합치기
  - basename/dirname/extname: 부분 추출
  - parse: 상세 분석
  - isAbsolute/relative/resolve: 경로 확인
  - isValidPath/isSubpath/commonPath: 고급 기능
  - 15개 메서드

**테스트**: 12/12 통과 ✅
