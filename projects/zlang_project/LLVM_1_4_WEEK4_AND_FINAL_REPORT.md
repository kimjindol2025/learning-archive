### 1. Hello World
```z
fn hello_world() {
    print("Hello, World!\n")
}
```

### 2. 계산기
```z
fn add(a, b) { a + b }
fn sub(a, b) { a - b }
fn mul(a, b) { a * b }
fn div(a, b) { a / b }
```

### 3. FizzBuzz (1-100)
```z
fn fizzbuzz(n) {
    let i = 1
    while (i <= n) {
        if ((i % 15) == 0) {
            print("FizzBuzz\n")
        }
        // ...
    }
}
```

### 4. 팩토리얼
```z
fn factorial(n) {
    if (n <= 1) { return 1 }
    else { return n * factorial(n - 1) }
}
```

### 5. 소수 판별
```z
fn is_prime(n) {
    if (n <= 1) { return 0 }
    if (n <= 3) { return 1 }
    // ...검사 로직...
}
```
