### App 2: fizzbuzz (루프 + 조건)

```z
fn fizzbuzz(n) {
    let i = 1
    while (i <= n) {
        if ((i % 15) == 0) {
            print("FizzBuzz\n")
        } else if ((i % 3) == 0) {
            print("Fizz\n")
        } else if ((i % 5) == 0) {
            print("Buzz\n")
        } else {
            print(i)
            print("\n")
        }
        i = i + 1
    }
}

fn main() {
    fizzbuzz(15)
}
```

**결과**: ✅ **PASS**
- 컴파일 시간: 67ms
- 바이너리 크기: 5.8KB
- 정확한 FizzBuzz 출력 (15줄)
- Exit code: 0

---

### App 3: factorial (재귀 함수)

```z
fn factorial(n) {
    if (n <= 1) {
        return 1
    } else {
        return n * factorial(n - 1)
    }
}

fn main() {
    let result = factorial(5)
    print("5! = ")
    print(result)
    print("\n")
}
```

**결과**: ✅ **PASS**
- 컴파일 시간: 48ms
- 바이너리 크기: 4.6KB
- 정확한 출력: "5! = 120"
- Exit code: 0
