# Changelog - FreeLang v4 Mobile

## [1.0.0] - 2026-02-20

### Added
- **AppBuilder**: 모바일 앱 구성 및 빌드
  - iOS/Android 크로스플랫폼 지원
  - 테마 시스템
  - 네이티브 API 접근

- **UIComponents**: 풍부한 UI 컴포넌트 라이브러리
  - Text, Button, TextInput, List, Image, Card
  - 반응형 레이아웃
  - 애니메이션 지원

- **HttpClient**: HTTP 클라이언트
  - GET, POST, PUT, DELETE 지원
  - Bearer 토큰 인증
  - 인터셉터 체인
  - 자동 재시도

- **Database**: SQLite 로컬 데이터베이스
  - 테이블 관리
  - CRUD 작업
  - 트랜잭션 지원

- **SyncManager**: 오프라인-온라인 동기화
  - 자동 변경사항 추적
  - 충돌 해결
  - 스마트 동기화

- **BiometricAuth**: 생체 인증
  - 지문 인식
  - Face ID 지원
  - 플랫폼별 구현

- **PushNotification**: 푸시 알림
  - 로컬 알림
  - 원격 알림
  - 예약 알림

### Performance
- 시작 시간: <2초 (콜드 스타트)
- 메모리: 50-100MB
- 배터리: <5% (1시간)
- 네트워크 지연: <100ms

### Platform Support
- iOS 13+
- Android 8.0+
- Cross-platform 100% 코드 공유
